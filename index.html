Permet by joergreichert

Exercising Graphiti

Playing around with Graphiti, Xtext and EEF

Dependencies

Eclipse 3.7 (Indigo) Graphiti 0.8 Xtext 2.0 EEF 1.0 GEF 3.7.0

Install

1. Download and extract Eclipse Indigo: http://www.eclipse.org/downloads/packages/eclipse-modeling-tools/indigor 2. Install Graphiti 0.8: http://download.eclipse.org/graphiti/updates/0.8.0 3. Install Xtext 2.0: http://download.eclipse.org/modeling/tmf/updates/releases/ 4. Extended Editing Framework: http://download.eclipse.org/releases/indigo 5. Show View Git Repositories, Clone, git://github.com/joergreichert/Permet.git 6. Pull 7. Run Configuration, New Eclipse Configuration, Run 8. Import example project FowlerDslTest in Runtime Workspace
